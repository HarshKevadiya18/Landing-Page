## 📋 Task Overview

Landing-Page/

---

## How to Run

1. Clone or download the project.
2. Open the folder in VS Code.
3. Install the **Live Server** extension.
4. Right-click on `index.html`.
5. Click **Open with Live Server**.
6. Open the generated URL in your browser.
7. Resize the browser window to test responsiveness.

---

## Responsive Design

The landing page adapts to different screen sizes using CSS Media Queries.

### Desktop View

* Horizontal navigation menu
* Large hero section
* Full-width layout

### Mobile View

* Navigation links stack vertically
* Smaller text sizes
* Optimized spacing and alignment
